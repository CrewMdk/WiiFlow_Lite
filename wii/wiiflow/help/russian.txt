Средства управления:
- Главное меню (Coverflow) :
-- 'Вверх' / 'Вниз' = Предыдущая / следующая игра (вертикально)
-- 'Влево' / 'Вправо' = Предыдущая / следующая игра (горизонтально)
-- 'Минус' / 'Плюс' = Быстрое пролистывание игр
-- 'Home' = Открыть домашнее меню / Выйти из WiiFlow
-- 'Home' (Sourceflow / меню) = Открыть конфигурацию меню источников
-- 'Home' (поток Homebrew) = Открыть конфигурацию потока Homebrew
-- '1' / '2' = Предыдущая / следующая раскладка Coverflow
-- 'A' = Выбрать игру
-- 'B' = Открыть меню источников (если оно установлено)
-- 'B'+'A' = Немедленно запустить игру
-- 'B'+'Влево' / 'B'+'Вправо' = Сменить музыкальный трек
-- 'B'+'Вверх' / 'B'+'Вниз' = Предыдущая / следующая игра в отсортированном списке (по умолчанию по алфавиту)
-- 'B'+'Минус' = Запуск или выбор случайной игры из текущего Coverflow
-- 'B'+'Плюс' = Выбрать тип сортировки (по алфавиту, количеству игроков, количеству запусков и т.д...)
-- 'Z' / 'ZR' = Сделать снимок экрана

-- 'A' на стрелках на экране = Быстрое пролистывание игр
-- 'B' на стрелках на экране = Предыдущая / следующая игра в отсортированном списке (по умолчанию по алфавиту)
-- 'A' на значке Звездочка = Показывать все / только избранное
-- 'A' на значке Категории = Выбрать категории для отображения
-- 'A' на значке Шестеренка = Главное меню настроек
-- 'A' на значке USB(Wii), GC, Каналы, Плагины или Homebrew = Переключиться на этот источник
-- 'A' на значке Дом = Открыть домашнее меню / Выйти из WiiFlow
-- 'A' на значке Диск (диск с игрой в дисководе) = Запустить диск с игрой

- Обложки игр:
-- 'A' на обложке = Перевернуть обложку
-- 'A' вне экрана = Запустить игру
-- 'A' на значке Звездочка = Избранное: добавить / удалить (желтый = избранное)
-- 'A' на значке Категории = Установить категории для текущей игры
-- 'A' на значке Рука = Родительская блокировка (красный = заблокировано)
-- 'A' на значке Шестеренка = Параметры игры
-- 'A' на значке X = Удалить игру
-- 'B' = Назад к Coverflow
-- 'Вверх' / 'Вниз' = Предыдущая / следующая игра (вертикально)
-- 'Влево' / 'Вправо' = Предыдущая / следующая игра (горизонтально)
-- 'Минус' = Воспроизведение игрового видео, если оно доступно
-- 'Плюс' = Информация об игре
---'Вправо' (в информации об игре) = Описание игры
---'Влево' (в описании игры) = Информация об игре

- Перевернутая обложка:
-- 'B' = Сбросить обложку
-- 'Вверх' / 'Вниз' / 'Влево' / 'Вправо' = Переместить обложку
-- 'Плюс' = Увеличение масштаба
-- 'Минус' = Уменьшение масштаба
 
- Любые меню настроек:
-- 'Минус' / 'Плюс' = Предыдущая / следующая страница
-- 'Влево' / 'Вправо' = Предыдущая / следующая страница
-- 'Вверх' / 'Вниз' = Предыдущая / следующая кнопка

- Настройка раскладок Coverflow:
-- 'В'+'Минус' / 'В'+'Плюс' = Предыдущая / следующая страница
-- 'B' = Более быстрая настройка ('B'+'A' вместо просто 'A' для нажатия кнопки)
-- 'B'+'1' = Скопировать весь Coverflow
-- 'B'+'2' = Вставить Coverflow
